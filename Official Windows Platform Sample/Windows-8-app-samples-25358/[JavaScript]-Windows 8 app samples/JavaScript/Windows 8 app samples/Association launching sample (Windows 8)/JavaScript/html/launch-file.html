Clicking on the Launch Default Handler button will launch a .png file with the default handler installed on the system.
Clicking on the Launch with Warning button marks the file as untrusted, which results in a warning prompt being shown prior to launching the file.
Clicking on the Launch Open With button will launch the 'Open With' dialog that lets the user chose the handler for the .png file.
Clicking on the Pick and Launch button will launch the file picker that lets the user pick the file to launch. The picked file is then launched.
All variations use Windows.System.launchFileAsync.


The default handler application for .png files is launched and displays in the foreground. This sample is moved to the background.
If there is no default handler for .png or the Launch Open With button is clicked an 'Open With' dialog is displayed.
