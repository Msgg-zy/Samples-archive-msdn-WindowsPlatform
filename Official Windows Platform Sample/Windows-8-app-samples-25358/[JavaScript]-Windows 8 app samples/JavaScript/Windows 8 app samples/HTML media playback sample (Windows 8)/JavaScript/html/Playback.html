Windows 8 provides rich media format support through the HTML5 media element, i.e. the video tag and the audio tag, which are built using the Windows Media Foundation technologies. This sample code demonstrates basic playback of various media formats through the video tag. You can test playback of each media fomat by clicking on the corresponding buttons below. You can also try to playback other media files (e.g. .wmv, .wma, .wav, etc.) you have on your system in the music and video libraries using the File Picker button.

A general recommendation for developing interoperable media experiences is to use standardized media formats in your web apps.
