Press and hold to learn
Press and hold an item to bring up detailed info or teaching visuals without invoking an action. The Tooltip control is the easiest way to implement this. This action might also bring up a context menu.
Tooltip
A tooltip is a small pop-up window that is used to tell users what an unlabeled control is or does (such as a back button arrow) or to reveal the rest of occluded text that ends with an ellipsis (…).
Infotip
An infotip shows more info than a tooltip, such as multiple columns of information and image thumbnails.
Similar to when you use a mouse and hover over an item.
