Tiles are the representation of an application in Start and can use tile and badge notifications to display new, relevant, and tailored content to the user.

Tile notifications are specified in XML, and contain text and images in a pre-defined layout.

Applications must always include a square logo in the manifest, but applications that want to use wide templates MUST include a wide logo as well. If an application supports a wide tile, it is STRONGLY RECOMMENDED to send both wide and square templates in one XML payload since users decide what size your application's tile will be.
