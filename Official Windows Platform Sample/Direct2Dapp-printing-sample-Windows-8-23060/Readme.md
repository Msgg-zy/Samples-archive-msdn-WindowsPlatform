## Description

<div id="mainSection">
<p>This sample demonstrates how to add <a href="http://msdn.microsoft.com/library/windows/apps/dd370990">
Direct2D</a> printing support to a Windows Store app. </p>
<p>This sample shows how to use <a href="http://msdn.microsoft.com/library/windows/apps/dd370990">
Direct2D</a> features to render the content of a Windows Store app for printing and how to send the rendered content to the printer.</p>
<p>Note that this sample describes printing and how to render content on the page at a very low level of abstraction and is intended for software developers who work with app content at the pixel level of detail.</p>
<p>These topics provide more info about the APIs used in this sample:</p>
<ul>
<li><a href="http://msdn.microsoft.com/library/windows/apps/dd370990">Direct2D</a>
</li><li><a href="http://msdn.microsoft.com/library/windows/apps/hh847997"><b>ID2D1PrintControl</b></a>
</li></ul>

<ol>
<li>Start Microsoft Visual Studio Express&nbsp;2012 for Windows&nbsp;8 and select <b>File</b> &gt;
<b>Open</b> &gt; <b>Project/Solution</b>. </li><li>Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Visual Studio Express&nbsp;2012 for Windows&nbsp;8 Solution (.sln) file.
