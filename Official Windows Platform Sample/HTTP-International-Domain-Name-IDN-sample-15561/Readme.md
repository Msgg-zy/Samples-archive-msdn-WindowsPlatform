## Description

<div id="mainSection">
<p>The sample demonstrates how to use <a href="http://msdn.microsoft.com/en-us/library/windows/desktop/hh447432">
<b>HttpPrepareUrl</b></a> to normalize an international domain name and retrieve a new URL which is safe and valid to use in other HTTP APIs.
