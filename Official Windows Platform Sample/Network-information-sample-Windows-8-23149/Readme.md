## Description

<div id="mainSection">
<p>This sample demonstrates how to use the Windows Runtime Network Information APIs to demonstrate the following:
</p>
<ul>
<li>Get a list of connected profiles on the network. </li><li>Get profile information for the Internet connected profile </li><li>Retrieve local data usage for the Internet Connection profile </li><li>Get LAN Identifiers </li><li>Register for Network Status Change notifications </li></ul>
<p>This sample is provided in JavaScript/C#/C&#43;&#43;/Visual Basic and requires some experience with networking concepts and development practices.</p>
