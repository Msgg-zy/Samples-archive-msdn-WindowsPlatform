Sample Description

This page illustrates several key file-access features: programmatically accessing files (such as to enumerate a folder), getting a file's properties, and writing data to and reading data from a sample file (created automatically when you perform the first "write").

Sample Instructions

Click the button associated with the part of the sample you wish to run:

  • Enumerate Pictures - Illustrates how to programmatically access files by enumerating and displaying all files in Pictures.
  • Get properties of all files in Pictures - For all files in Pictures, displays top-level and basic properties.
  • Write text to a sample file - Illustrates writing data to a file by writing the current timestamp to a file called FileHandlingJs.txt in Pictures. (This file is automatically created if it does not exist already.)
  • Read text from a sample file - Illustrates reading data from a file by reading and displaying the contents of a file called FileHandlingJs.txt in Pictures.
