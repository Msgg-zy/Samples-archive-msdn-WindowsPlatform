Sample Description

Pickers display a list of files or folders from which a user can select one or more items for further processing. This page illustrates several ways to use the various pickers: selecting a single file, selecting multiple files, and selecting a folder.

Sample Instructions

Click the button associated with the part of the sample you wish to run:

  • Select a single file - Displays a FileOpenPicker allowing the selection of a single file.
  • Select multiple files - Displays a FileOpenPicker allowing the selection of multiple files.
  • Select a folder - Displays a FolderPicker allowing the selection of a single folder.
