Feature Description

In Windows 8.1, users are able to mark OneDrive files as either available-offline (default) or online-only. This capability enables users to move large files (such as pictures and videos) to their OneDrive, mark them as online-only, and save disk space (as a file containing only metadata about the remote file is kept locally).

Sample Description

Get all the files in Pictures with their storage provider (This PC, OneDrive, Network, etc.) and indicate whether they are currently available.

Sample Instructions

To test listing files that are on OneDrive, but unavailable, perform the following steps:

  1. Open the OneDrive app.
  2. Right-click the app, then click "New folder".
  3. Name the folder something like "Pictures on OneDrive", then click "Create".
  4. Click newly created folder to open it.
  5. Right-click anywhere on the page, then click "Add files".
  6. Browse to any local folder and select the desired file(s) to upload.
  7. Click "Copy to OneDrive".
  8. Right-click the newly added-to-OneDrive file(s), then click "Make online-only".
  9. When the confirmation dialog appears, click "Make online-only".
  10. Open a File Explorer window.
  11. Click on OneDrive (left panel).
  12. Right-click the folder you created earlier, then (from the context menu) click "Include in library->Pictures".
  13. Disconnect from OneDrive (e.g., unplug your ethernet cable or disable the network adapter).
  14. Click "Display Availability of Files in Pictures" below. The OneDrive file(s) you added earlier should be listed as "not available".
