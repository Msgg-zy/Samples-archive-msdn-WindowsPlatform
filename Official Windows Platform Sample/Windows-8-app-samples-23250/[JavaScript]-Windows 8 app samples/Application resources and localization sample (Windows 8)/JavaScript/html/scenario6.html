It is possible that, while an application is running, the language, scale, contrast or other settings may change. In order to handle these events, event listeners should be registered to listen for and react to the change.

To see new resources, change the the language by going to the Desktop Control Panel > Language > Add a language that is supported by the app (de-DE, en-US, fr-FR, or ja) > Move the new language to the top of the list.
