Clicking on the Launch Default Handler button will launch the URI http://www.bing.com with the default handler installed on the system.
Clicking on the Launch with Warning button marks the URI as untrusted, which results in a warning prompt being shown prior to launching the URI.
Clicking on the Launch Open With button will launch the 'Open With' dialog that lets the user chose the handler for the URI.
All variations use Windows.System.launchUriAsync.

The default handler application for http is launched and displays in the foreground. This sample is moved to the background.
If there is no default handler for http or the Launch Open With button is clicked an 'Open With' dialog is displayed.
