This sample demonstrates how to call Windows.Security.Authentication.OnlineId APIs to authenticate a user with Microsoft Account and use JWT tokens.

To setup the sample server that parsers JWT token, navigate to the Server folder in MicrosoftAccount samples and change the Web.config file.
When you register your application at https://manage.dev.live.com/Applications specify the values of 'Client ID', 'Client secret' and 'Redirect domain' in Web.config file.
The C# sample is configured to work with the sample server.

To sign out from the application, go to the Sessings charms.
