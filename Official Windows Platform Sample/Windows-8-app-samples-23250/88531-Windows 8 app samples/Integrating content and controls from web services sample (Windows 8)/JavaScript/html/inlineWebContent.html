An application can include a local page with inline web content. In the example below, a Bing Maps control is embedded in a local page. This page must be loaded in the web context (ms-appx-web://) in order to reference remote scripts.

The Bing API requires a AppID key to identify the application. To use this sample without warnings in the map, you will need to get a key from the Bing Developer Center. Then paste the key into the text box below.
