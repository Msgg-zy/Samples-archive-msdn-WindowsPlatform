To load in content from a separate HTML document, place the following CSS rule on an IFRAME element: -ms-flow-into: content;This will load the HTML document that is specified in the IFRAME's srcattribute.

To declare CSS Regions to accept content, place the following CSS rule on any DIV element: -ms-flow-from: content;This will flow the HTML content loaded by the IFRAME element and distribute it over the CSS Region elements.
