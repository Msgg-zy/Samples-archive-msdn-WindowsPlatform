This sample demonstrates how a developer can use the new CSS View State Media Query feature to detect Application Views State.

Rotate the screen or device to place the sample application in fullscreen-landscape to fullscreen-portrait.

Snap and unsnap the application to place the sample application in snapped or in fullscreen-landscape.

Snap another application to place the sample application in filled.
