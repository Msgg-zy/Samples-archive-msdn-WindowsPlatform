This sample app shows a basic example of activation. It shows how to save state and retrieve it after re-activation. State is saved in sessionState which is persisted when the application is suspended. State can be retrieved from sessionState when the application is re-activated.

To simulate suspending this application from Microsoft Visual Studio 11 Express, first launch it using "Start Debugging" and then use the "Suspend" button.




Fill in the text boxes below and simulate suspending this application by clicking on the "Suspend" button in Visual Studio 11 Express. Upon suspension, the filled in text is saved. After suspension the application can be terminated by clicking on the "Suspend and shut down" button. On re-launch, the text boxes will contain the text previously entered when the application was suspended.
