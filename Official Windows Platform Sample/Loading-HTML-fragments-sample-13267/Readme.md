## Description

<div id="mainSection">
<p>This sample demonstrates how to use <a href="http://msdn.microsoft.com/library/windows/apps/br229781">
WinJS.UI.Fragments</a> to load HTML defined on another page. </p>
<p>It shows: </p>
<ul>
<li>How to perform basic fragment loading. </li><li>How to perform custom fragment loading. </li><li>How to load fragments that contain scripts and Cascading Style Sheets (CSS) styles.
</li><li>How to load fragments that contain Windows Library for JavaScript controls. </li></ul>
