Getting started with C# and Visual Basic: Hello World samples

 

This sample includes the companion code for the Microsoft Visual Basic and C# tutorials in the Getting started node of the Windows Dev Center.

 

Specifically, this sample includes code for:

Part 1: Create a "Hello, world!" app This sample and tutorial demonstrate how to create a simple "Hello, world" Windows Store app using Extensible Application Markup Language (XAML) with Visual Basic or C#.

Part 2: Manage app lifecycle and state  This sample and tutorial update the "Hello, world" app to respond to lifecycle events, and save app and session data.

Part 3: Navigation, layout, and views This sample and tutorial update the "Hello, world" app with a photo page that adapts to different layouts and views.

Part 4: File access and pickers This sample and tutorial update the "Hello, world" app to let the user pick a photo from their library to display in the photo page.

This sample is written in XAML. For the HTML version, see the Getting started with JavaScript: Hello World samples.

To obtain an evaluation copy of Windows 8.1 Preview, go to Window 8.1 Preview.

To obtain an evaluation copy of Visual Studio 2013 Preview, go to Visual Studio 2013 Preview.

Note:  For Windows 8 app samples, download the Windows 8 app samples pack. The samples in the Windows 8 app samples pack will build and run only on Microsoft Visual Studio 2012.

Build the sample

  1. Start Visual Studio 2013 Preview and select File > Open > Project/Solution.

  2. Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Visual Studio 2013 Preview Solution (.sln) file.

  3. Press F7 or use Build > Build Solution to build the sample. Run the sample

  4. This Visual Studio 2013 Preview solution includes four separate projects. To set the project that you want to run as the startup project, right-click the solution in Solution Explorer and select Set as StartUp Project.

    To run the app and then debug it, press F5 or use Debug > Start Debugging. To run the app without debugging, press Ctrl+F5 or use Debug > Start Without Debugging.
