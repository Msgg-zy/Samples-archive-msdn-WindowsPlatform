The sample demonstrates some basic principles of Windows Store app development in native C++ using XAML to define the user interface. It is a complete working version of the application discussed on the Windows Developer Center in the article entitled Create your first app in C++. The app is a blog reader that downloads the RSS and Atom feeds for several Windows-related blogs. You can easily modify it to point to your own blogs of choice. This sample introduces C++ projects for Windows Store appsin Visual Studio, and shows how to use the blank project template, the split page, the grid page, and the basic page. The sample also shows how the C++ Component Extensions(Visual C++ component extensions (C++/CX)) enable you to create Windows Store apps in a modern C++ idiom rather than a traditional COM programming style. The sample covers the basics of XAML, including commonly used elements such as Grid , StackPanel , TextBlock , and ListView . It uses custom styles and animations to make a distinctive user interface, and has an app bar. It also demonstrates how to use databinding to update user interface elements from user-defined properties and collections, how to format Windows::Foundation::DateTime values, and how to make asynchronous method calls using the concurrency::task class .

Build the sample

  1. Start Visual Studio Express 2012 for Windows 8 and select File > Open > Project/Solution.
  2. Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Visual Studio Express 2012 for Windows 8 Solution (.sln) file.
  3. Press F7 or use Build > Build Solution to build the sample.

Run the sample

To debug the app and then run it, press F5 or use Debug > Start Debugging. To run the app without debugging, press Ctrl+F5 or use Debug > Start Without Debugging.
