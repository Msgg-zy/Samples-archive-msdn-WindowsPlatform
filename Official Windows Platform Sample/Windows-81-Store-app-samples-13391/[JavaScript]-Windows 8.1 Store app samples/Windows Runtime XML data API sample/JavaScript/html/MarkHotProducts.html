This sample demonstrates how to use the XML APIs in the Windows Runtime to traverse a Document Object Model (DOM) tree to update the data and save it to a file.

To run the sample, click the Mark Hot Products button. The ‘hot’ attribute will be updated to ‘1’ if the value of the ‘sell10days’ attribute is greater than the value of the ‘InStore’ attribute. Click the Save button to save the XML contents.
