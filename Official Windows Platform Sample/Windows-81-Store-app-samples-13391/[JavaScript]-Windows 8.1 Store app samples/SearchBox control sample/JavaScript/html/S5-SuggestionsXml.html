This example shows how to supply suggestions to the Search box from a web server that returns suggestions in the XML Search Suggestions format.

Please follow these steps to try it out:

  1. Enter the URL to a web service that returns suggestions in the OpenSearch Suggestions Format. Use {searchTerms} in the URL to represent where you want the query to be inserted
  2. Enter a query into the Search box
  3. Suggestions will be provided
  4. Select a suggestion or submit the text in the search box by pressing the search button
