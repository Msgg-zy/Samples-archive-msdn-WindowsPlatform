Use linguistic alternatives to query terms for East Asian search (primarily in China and Japan). The alternatives are matched against a static set of suggestions. Note: A Japanese IME must be enabled for this scenario to function properly.

Please follow these steps to try it out:

  1. Type in a query such as 横浜市 using a Japanese IME
  2. Suggestions will be provided
  3. Select a suggestion or submit the text in the search box by pressing the search button
