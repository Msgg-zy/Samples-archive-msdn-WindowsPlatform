Type a Japanese text in the first text box and press "Start" button. The text will be split into segments separated by "/" in the second text box based on the result of JapanesePhoneticAnalyser.getWords(). The unit to split the text based on can be changed by "Use mono ruby" option. If the option is unchecked, each segments consists of words with the reading. If the option is checked, each segments get split not into words but into character tokens whose reading is atomic.
