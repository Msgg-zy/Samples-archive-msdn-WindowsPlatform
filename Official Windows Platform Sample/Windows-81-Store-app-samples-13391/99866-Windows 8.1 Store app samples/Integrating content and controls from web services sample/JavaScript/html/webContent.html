An application can include an existing web page by loading the web page in an x-ms-webview element. An x-ms-webview that navigates to a HTTP or HTTPS URI will use a web context, which does not allow the x-ms-webview to access Windows Runtime APIs. If packaged content in your application is loaded with the MS-APPX:// URI scheme, it will use a local context and is allowed to access WinRT APIs from this context.
