When a video element is in the optimized rendering mode, the stereo video scenario can be enabled when the system is stereo-capable and the display setting is turn on for stereo rendering. The video element has been extended to provide the interfaces for Windows Store apps to include the stereo video playback experience. The Windows Store app platform supports frame-compatible formats with side-by-side and top-bottom frame packing modes, and it relies on the standard-compliant medtadata (e.g. H.264 Frame Packing SEI) in the video files to determine the frame packing options. By default, the platform will show a mono view of the stereo video even when the system is capable of stereo display.
