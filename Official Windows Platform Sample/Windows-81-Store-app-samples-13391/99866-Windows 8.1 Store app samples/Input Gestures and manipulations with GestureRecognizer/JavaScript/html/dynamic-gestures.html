The GestureRecognizer supports both static and dynamic gestures. Static gestures are single finger interactions that make the UI respond in some way (e.g. tapping an element). Dynamic gestures enable manipulation of content through multi-touch interactions, allowing content to be pinched, stretched, rotated, or panned. In this example, the maze can be manipulated and the ball responds to the tap, double tap and press-and-hold gestures. Press the Start button to begin.
