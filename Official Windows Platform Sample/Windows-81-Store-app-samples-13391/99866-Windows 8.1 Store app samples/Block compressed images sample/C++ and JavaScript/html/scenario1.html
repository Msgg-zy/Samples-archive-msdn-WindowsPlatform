Use images with block compression (DDS images) in a Windows Store JavaScript app. DDS images can be used like any other format like JPEG or PNG; for example in an Image tag or in a Canvas 2D Context.

IMPORTANT: Before running this JavaScript project, you must first do the following:

  1. Build the ImageContentPipeline C++ project. This converts the images in \OriginalAssets to block compressed (DDS) form in \BlockCompressedAssets.
  2. In Solution Explorer, right click on the BlockCompressedAssets folder > Add > Existing Item.
  3. Navigate to the BlockCompressedAssets folder and add the two DDS images.

Refer to readme.txt located in the root of the solution for more information.
