Application Edge UI can be invoked using touch, mouse, or keyboard.

1. Touch: Drag from the top or bottom edge of the screen. The following events are supported by the Edge Gesture API:

(a) "starting" is triggered when the drag begins;
(b) "completed" is triggered when the user lifts his or her finger;
(c) "canceled" is triggered when the user either returns to the edge where the drag began or continues dragging into the middle of the screen.

2. Mouse: Right-click anywhere in the sample. This will trigger the "completed" event.

3. Keyboard: Press Win+Z or the context menu key. This will trigger the "completed" event.
