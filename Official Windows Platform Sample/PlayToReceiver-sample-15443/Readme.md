## Description

<div id="mainSection">
<p>This sample demonstrates how to create a software Play To receiver. </p>
<p>To advertise the software Play To Receiver, click the <b>Start Receiver</b> button. To stop the receiver, click the
<b>Stop Receiver</b> button. After the software Play To receiver is advertised on the network, you can stream media from a different PC on your local network to the software receiver. This sample will only work if the Play To source video or audio being received
 is from a different computer.</p>
<p>To obtain an evaluation copy of Windows&nbsp;8.1, go to <a href="http://go.microsoft.com/fwlink/p/?linkid=301696">
Windows&nbsp;8.1</a>.</p>
<p>To obtain an evaluation copy of Microsoft Visual Studio&nbsp;2013, go to <a href="http://go.microsoft.com/fwlink/p/?linkid=301697">
Visual Studio&nbsp;2013</a>.</p>
