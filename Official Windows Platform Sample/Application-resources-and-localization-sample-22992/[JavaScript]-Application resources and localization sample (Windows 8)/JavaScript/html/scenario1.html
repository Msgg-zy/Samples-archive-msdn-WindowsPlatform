Images and other file resources can be loaded based on scale factor, high contrast mode, user language and other dimensions. All file references in HTML and code can be virtualized to pick up the most appropriate image based on the current context of the application.

Note that each of the img tags in the output references a logical image (images/simpleprojector.png), but loads a physical image in the package (images/simpleprojector.scale-100.png). The physical files can be qualified with folder names (e.g. en-US/) and/or filenames (e.g. name.contrast-high.png).

To see a different image, follow the instructions below and re-run the application.

  • To change the user language preference, Desktop Control Panel > Clock, Language, and Region > Language > Add a language that is supported by the app (de-DE, en-US, fr-FR, or ja) > Move the new language to the top of the list.
  • To change the scale factor, use the Visual Studio simulator
  • To change contrast, go to PC settings > Ease of Access > Turn on "High Contrast"
