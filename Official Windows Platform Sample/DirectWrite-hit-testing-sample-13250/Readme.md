## Description

<div id="mainSection">
<p>This sample shows how to use the hit-testing features of <a href="http://msdn.microsoft.com/library/windows/apps/dd368038">
DirectWrite</a> to determine which part of the displayed text is being clicked or touched.
</p>
<p>These topics provide more info about the APIs used in this sample:</p>
<ul>
<li><a href="http://msdn.microsoft.com/library/windows/apps/dd368038">DirectWrite</a>
</li><li><a href="http://msdn.microsoft.com/library/windows/apps/dd370990">Direct2D</a>
</li><li><a href="http://msdn.microsoft.com/library/windows/apps/dd316718"><b>IDWriteTextLayout</b></a>
</li><li><a href="http://msdn.microsoft.com/library/windows/apps/br208225"><b>Windows::UI::Core::CoreWindow</b></a>
</li></ul>

<h2>Build the sample</h2>
<ol>
<li>Start Visual Studio&nbsp;2013 and select <b>File &gt; Open &gt; Project/Solution</b>.
</li><li>Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Microsoft Visual Studio Solution (.sln) file.
