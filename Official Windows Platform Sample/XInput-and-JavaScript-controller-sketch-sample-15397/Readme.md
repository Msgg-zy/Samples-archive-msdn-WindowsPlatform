## Description

<div id="mainSection">
<p>This sample demonstrates how to wrap the XInput C&#43;&#43; API in a Windows Runtime component and call it from a Windows Store app using JavaScript.
</p>
<p>This sample implements a sketch app that lets you use the Xbox game controller to select line thickness and draw on the screen. Specifically, it features:</p>
<ul>
<li>A hybrid app design that uses a C&#43;&#43; Windows Runtime type that wraps XInput </li><li>Calls to the Windows Runtime XInput object from JavaScript </li><li>HTML5 Canvas drawing </li></ul>
<p>These topics provide more info about the APIs used in this sample:</p>
<ul>
<li><a href="http://msdn.microsoft.com/library/windows/apps/ee417001">XInput</a> </li><li><a href="http://msdn.microsoft.com/library/windows/apps/br226021"><b>Windows.Foundation</b></a>
</li></ul>
<p>This sample requires some familiarity with C&#43;&#43; and the Windows Runtime.</p>

<h2>Build the sample</h2>
<ol>
<li>Start Visual Studio&nbsp;2013 and select <b>File</b> &gt; <b>Open</b> &gt; <b>Project/Solution</b>.
</li><li>Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Visual Studio&nbsp;2013 Solution (.sln) file.
</li><li>Press F7 or use <b>Build</b> &gt; <b>Build Solution</b> to build the sample. </li></ol>
<h2>Run the sample</h2>
<p>To debug the app and then run it, press F5 or use <b>Debug</b> &gt; <b>Start Debugging</b>. To run the app without debugging, press Ctrl&#43;F5 or use
<b>Debug</b> &gt; <b>Start Without Debugging</b>. </p>
<p>To use this sample, connect an Xbox 360 common controller then use the left and right thumbsticks to control the movement of the drawing pointer. Press the right trigger to change the shape of the pointer.</p>
