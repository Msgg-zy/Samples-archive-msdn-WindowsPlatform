In addition to demonstrating gestures (Pen, Mouse and Touch) on a tile - Transformations, Tap to change colors, Press and Hold for restoring original form, this sample showcases Mouse Wheel Support. Using scroll on the mouse - one can translate the tile. Using Shift + Ctrl + scroll - one can Rotate the tile (Origin is the mouse pointer head). Using Ctrl + scroll - one can perform scale operations on the tile.

Press the Start button to start.
