## Description

<div id="mainSection">
<p>This sample shows how to use the Windows.Devices.Sensors.OrientationSensor API.</p>
<p>This sample allows the user to view the rotation matrix and Quaternion values that reflect the current device orientation. You can choose one of two scenarios:
</p>
<ul>
<li>Orientation sensor data events </li><li>Current orientation sensor reading </li></ul>
<p></p>
<p><b>Orientation sensor data events</b></p>
<p>When you click the <b>Enable</b> button for the <b>Orientation sensor data events</b> option, the app begins streaming sensor readings in real time.</p>
<p><b>Current orientation sensor reading </b></p>
<p>When you click the <b>Get</b> button for the <b>Current orientation sensor reading</b> option, the app retrieves the most recent sensor reading.</p>
