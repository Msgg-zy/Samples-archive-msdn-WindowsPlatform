When you set the sandbox attribute with the allow-forms flag, the iframe content can submit forms but can’t execute scripts, or navigate the top level document. The content is also forced into a unique origin that prevents it from accessing content from the same origin, such as cookies.

These two iframe elements demonstrate the effects of sandboxing content. They both display the same example content, but one uses the sandbox attribute with the allow-forms flag and one doesn’t.
