## Description

<div id="mainSection">
<p>This sample demonstrates the use of XAudio2 in a app. </p>
<p>Specifically, this sample covers:</p>
<ul>
<li>Parsing a RIFF audio file for the audio data </li><li>Setting up XAudio2 to play a sound via a buffer </li><li>Playing and monitoring the playstate of a sound buffer in XAudio2 </li></ul>
<p></p>
<p>This sample is written in C&#43;&#43; and requires some experience with graphics programming.</p>
<p>For more info about the concepts and APIs demonstrated in this sample, see:</p>
<ul>
<li><a href="http://msdn.microsoft.com/library/windows/apps/hh405049">XAudio2</a>
</li></ul>
