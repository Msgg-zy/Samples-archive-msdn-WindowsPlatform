Guidelines for the introductory home page of your app.

Purpose

  • Ensure consistency of content across app and corresponding gallery.xml file.
  • Increase the usefulness of the content in app and corresponding gallery.xml file.

Required

  • Minimum of 1,000 characters
  • Title and abstract with SEO keywords
  • List of features demonstrated by the sample
  • Instructions for building and the sample
  • Links to related topics (minimum of 3)
  • Programming language-specific callouts

Optional

  • How to use the sample
  • Opportunities to extend the sample
  • Design guidance
  • Best practices
  • Code callouts
  • Screenshots
  • Byline
