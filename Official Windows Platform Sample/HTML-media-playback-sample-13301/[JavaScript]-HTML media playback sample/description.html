This sample demonstrates how to play audio and video using the HTML5 video tag in a Windows Store app using JavaScript.

Specifically, this sample shows how to:

  • Play a video using the video tag. For more info about using the video tag to play video, see Quickstart: playing video in a web app.
  • Use the audioTracks property to choose between different language tracks in a video that has multiple language tracks.
  • Use msZoom to remove letterbox or pillarbox from a video.
  • Display subtitles during video playback using the "track" element and "text track model". This sample code shows how to use a WebVTT file.
  • Enabling Stereo 3D playback. If a system is stereo-capable, the stereo video scenario can be enabled when a video element is in the optimized rendering mode and the display setting is set for stereo rendering.

Build the sample

  1. Start Visual Studio 2013 and select File > Open > Project/Solution.
  2. Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Visual Studio 2013 Solution (.sln) file.
  3. Press F7 or use Build > Build Solution to build the sample.

Run the sample

To debug the app and then run it, press F5 or use Debug > Start Debugging. To run the app without debugging, press Ctrl+F5 or use Debug > Start Without Debugging.
