## Description

<div id="mainSection">
<p>This sample demonstrates how to use <a href="http://msdn.microsoft.com/library/windows/apps/hh701658">
<b>ApplicationView</b></a> properties, such as orientation and adjacent edges, to make layout decisions.
</p>
<p>Specifically, this sample covers:</p>
<ul>
<li>Using window orientation, either landscape or portrait, to decide whether UI elements are placed horizontally or stacked vertically.
</li><li>Using edge information to decide where to position a control. The edge information specifies if the app is:
<ul>
<li>adjacent to the left edge of the display </li><li>adjacent to the right edge of the display </li><li>adjacent to both edges (full screen) </li><li>adjacent to neither edge (between two apps) </li></ul>
</li></ul>
